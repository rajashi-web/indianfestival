== Description ==
Rasti Lite is a nice , clean and Beautiful blogging WordPress theme 

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Rasti Lite in the search form and press the 'Enter' key in your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
