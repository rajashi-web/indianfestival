== Description ==
A very simple plugin to add portfolio - The Most creative Mobile Touch Slider - for WordPress. 

Check [Demo and Features](https://demo.wponlinesupport.com/portfolio-and-projects-demo/) for additional information.

**Also work with Gutenberg shortcode block.**

Portfolio/ Projects - is the free and most modern mobile touch slider with hardware accelerated transitions and amazing native behavior. It is intended to be used in Desktop and mobile websites.

= This plugin contain shortcode =

<code>[pap_portfolio]</code>

= Here is Template code =

<code><?php echo do_shortcode('[pap_portfolio]'); ?></code>

= Features include =

* Also work with Gutenberg shortcode block.
* Fully responsive
* Touch-enabled Navigation.
* Slide and fade effect
* Easy to install
* Given shortcode and template code.
* each project has screenshot slider.

= You can use Following parameters with shortcode =

* **limit:** limit="5" (Number of projects you want to show.by default is 5.)
* **category:** category="5,15,8" (display portfolio category wise.)
* **orderby:** orderby="date" (Display projects in your order. Values are "date", "modified", "title", "name" (Post Slug), "ID", "rand", "menu_order", "comment_count")
* **Order :** order="DESC" (Controls post order. Values are "ASC" OR "DESC".)
* **Grid :** grid="2" (Display post in Grid formats. You can use grid:1,2,3,4).
* **Link :** link="true" (Display portfolio link in popup or not).

== Installation ==
1. Upload the 'portfolio-and-projects' folder to the '/wp-content/plugins/' directory.
2. Activate the "portfolio-and-projects" list plugin through the 'Plugins' menu in WordPress.
